1. **安装元件库**：首先，解压下载的"ElementUI元件库.zip"文件。然后，在AxureRP中，依次点击“文件”>“库”>“导入”，选择解压缩后的.rp文件进行加载。
2. **开始设计**：导入成功后，您可以在Axure的库面板找到新增的Element UI元件库。直接拖动所需的组件到画布开始您的原型设计。
